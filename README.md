### 2. Dynamic Rotating Nebula (Sketch_2_Nebula)

**Original Sketch Name:** *Based on Generative Art principles*

**Modifications:**
* **Canvas & Color:** Set to `800x600` with an HSB color mode, utilizing a dark background and a two-color interpolation (magenta to cyan) for the particles.
* **Animation:** Implemented a **3D perspective effect** using Z-depth to simulate a rotating particle cloud or nebula.
* **Interaction:** The **mouse X-position** directly controls the rotation speed of the entire cloud, making the sketch interactive.
* **Variables:** Explicit variables (`NUM_POINTS`, `RADIUS`) were defined to easily control the density and size of the nebula.
